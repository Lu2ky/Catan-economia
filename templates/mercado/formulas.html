Marco economico

Reglas del mercado con regresion lineal

Los recursos se compran y venden con dinero. El precio de cada recurso se ajusta segun la demanda observada y el modelo lineal definido en la propuesta.

Demanda estimada

Q estimada = a + bP

a representa demanda base y b la sensibilidad ante cambios de precio.

Demanda real por operacion

Compra: +cantidad | Venta: -cantidad

Las compras suben demanda efectiva y las ventas la reducen.

Comparacion de demanda

Delta Q = Q real - Q estimada

Si Delta Q > 0 hay presion de demanda; si Delta Q < 0 el mercado se enfria.

Ajuste de precio

P nuevo = P actual + k * Delta Q

k es el factor de ajuste configurado en cada operacion.

Recalculo del modelo

Cada 5 turnos por recurso

En los turnos 5, 10, 15... se recalculan a y b con regresion lineal simple.

Validaciones monetarias

Saldo e inventario deben alcanzar

Para comprar se exige saldo; para vender se exige inventario suficiente.

Recomendaciones de uso

  • Configura primero jugadores y precios base de recursos.
  • Registra compras y ventas en la vista Mercado para alimentar la regresion.
  • Usa transferencias para pagos directos entre jugadores.
  • Revisa historial para analizar comportamiento de precios por turnos.
